142. Binary Matrix Optimization

Level: Hard Topic: Dynamic Programming

Given an m x n binary matrix mat, find the maximum side length of a square which is entirely composed of 1s after performing at most k row swaps.

Example 1:

Input: mat = [[0,1,1],[1,1,1],[0,1,1]], k = 1
Output: 3
Explanation: Swap row 0 and row 2.

Constraints:

  • m == mat.length
  • n == mat[i].length
  • 1 <= m, n <= 300
  • 0 <= k <= m
